# Check skill level by project runner offers based on ReactJS

## Dependencies

* node.js
* webpack
* react
* redux

## Installation

```
git clone https://github.com/oZoon/ndr.git
npm i
```

## Scripts
```
npm run devserv (development)
npm run prd (production)
```
